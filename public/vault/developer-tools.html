► Ferramentas para Jogos ​

► Geral ​

  • 🌟 LumaPlay - Execute jogos que dependem da Uplay da Ubisoft.
  • CapFrameX - Utiliza o overlay do RivaTuner para capturar o framerate durante o gameplay. O RTSS é apenas um provedor, não sendo instalado completamente no seu computador.
  • Simplaza-org - Site para download de addons, aeronaves e cenários para o Microsoft Flight Simulator 2020 (MSFS2020).
  • Ludusavi - Ferramenta para backup dos dados salvos de seus videogames para PC, escrita em Rust e disponível em várias plataformas.
  • Borderless Gaming - Jogue seus jogos favoritos em uma janela sem bordas; elimina a necessidade de Alt-Tabs demorados.
  • Rclone - Responsável por transferir seus saves de jogos crackeados para a nuvem, operando via linha de comando.
  • ProtonDB - Banco de dados de relatórios sobre a compatibilidade de jogos com sistemas operacionais Linux. Leitura recomendada
  • Better xCloud - Userscript para melhorar a experiência de jogos na Xbox Cloud e Remote Play em navegadores.
  • Smart Online Fix - Ferramenta para iniciar e organizar jogos do online-fix-me no Linux.

► Ferramentas para Steam ​

  • 🌟 SteamDB - Fornece informações de DLC para ferramentas como CreamAPI, GreenLuma e SmartSteamEmu.
  • Steam Content Packager - Ferramenta para empacotar seus arquivos de jogo do Steam, removendo informações de identificação para compartilhamento no subfórum Steam Content Sharing do cs.rin.ru.
  • Goldberg Emulator - Emulador Steam que permite crackear seu próprio jogo Steam ou arquivos Steam limpos compartilhados.
  • SmartSteamEmu, GreenLuma e CreamAPI - Emuladores e desbloqueadores de DLC para jogos Steam.
  • Auto-CreamAPI - Instala e configura automaticamente o CreamAPI para jogos Steam.
  • Steamless - Remove o DRM SteamStub dos executáveis de jogos Steam, para uso com emuladores Steam.

► Ferramentas para a Epic Games ​

  • ScreamAPI - Desbloqueador de DLC para o Epic Online Services, mantendo funcionalidades como multijogador e conquistas.
  • Free-games-claimer - Ferramenta para reivindicar automaticamente jogos gratuitos da Epic Store.

► Emuladores ​

  • Wii U USB Helper e Wii U Downloader - Ferramentas para baixar jogos do Cemu e de servidores da Nintendo.
  • Winlator@Frost - Versão modificada do emulador de Wine para Android com funcionalidades adicionais.
  • RetroReversing e RetroSix Wiki - Recursos para engenharia reversa e reparo de consoles retrô.
  • The Cutting Room Floor - Dedicado a explorar conteúdo removido ou não utilizado em jogos antigos.

Mais ferramentas para jogos disponíveis no subfórum de lançamentos do cs.rin.ru.

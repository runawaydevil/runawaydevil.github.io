# # # Por que impressionante LLM Apps?

- Descubra formas práticas e criativas que os LLMs podem ser aplicados em diferentes domínios, desde repositórios de códigos até caixas de e-mail e muito mais.
- Explorar aplicativos que combinam LLMs de OpenAI, Anthropic, Gemini e alternativas de código aberto com AI Agents, Agent Teams, MCP & RAG.
- Aprender com projetos bem documentados e contribuir para o crescente ecossistema de código aberto de aplicações LLM.

# # # Começar

1. **Clone o repositório**

```bash
git clone https://github.com/Shubhamsaboo/awesome-llm-apps.git
```

2. **Navegar para o diretório de projeto desejado**

```bash
cd awesome-llm-apps/starter_ai_agents/ai_travel_agent
```

3. **Instale as dependências necessárias**

```bash
pip install -r requisitos.txt
```

4. **Siga as instruções específicas do projeto** no arquivo `README.md` de cada projeto para configurar e executar o aplicativo.
